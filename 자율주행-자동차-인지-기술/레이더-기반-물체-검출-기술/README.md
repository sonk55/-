# 레이더 기반 물체 검출 기술

## Question
### 주변 방해 신호의 영향은 레이더의 오탐률을 높이는 주된 원인으로 알려져 있다. 방해 신호에 대해서 일정한 물제 검출 성능를 유지하려면 어떻게 해야하나?

## Answer
### 물체 검출을 위해 임계값을 주변의 방해 신호 에너지에 따라 조절해 주는 CFAR 검출 방법을 사용한다.
