# 카메라 보정 기술

## Question 
카메라 영상에서 물체의 위치와 실제 물체의 위치간의 관계를 알 수 있을까요?

### Answer : 카메라 캘리브레이션이라는 과정을 통해 실제 3차원 좌표계와 카메라 영상의 좌표계 간의 관계를 알아낼 수 있다.

---

## 학습목표
1. 카메라 좌표계에 대해 설명할 수 있다.
2. 카메라 내부 파라미터에 대해 설명할 수 있다.
3. 카메라 캘리브레이션에 대해 설명할 수 있다.


## 카메라 좌표계
* 카메라에서 물체를 표현하는 좌표계

## 카메라 내부 파라미터
* 카메라 영상 좌표계와 3차원 월드 좌표계 변환을 위한 카메라 내부의 기계적인 셋팅에 관련된 파라미터
* 초점거리 : 렌즈 중심과 이미지 센서와의 거리 (픽셀단위)
* 주점 : 렌즈의 중심에서 이미지 센서에 수직으로 내린 점의 영상 좌표(픽셀단위)
* 비대칭 계수 : 이미지 센서의 Cell Array의 Y축이 기울어진 정도

## 카메라 캘리브레이션
* 카메라 영상 좌표계와 3차원 월드 좌표계 사이의 변환 행렬을 알아내는 과정
* 자율주행차가 월드 좌표계에서의 물체 상태나 환경을 카메라 영상을 이용하여 이해하기 위해서는 카메라 캘리브레이션이 수행되어야 함
* 두개의 카메라를 이용해 거리를 알아낼때 카메라 캘리브레이션이 매우 중요

1. 반복적인 패턴이 있는 체커보드를 카메라로 촬영
2. 영상에서 체커보드의 코너점들을 검출
3. 결과를 이용하여 카메라의 내부 파라미터값 계산
